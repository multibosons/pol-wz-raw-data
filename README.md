# Doubly polarised WZ inclusive production at LHC

Raw data produced for p p -> W+(e+ nu_e) Z(mu+ mu-) + X @ NNLOQCD + NLOEW accuracy.

## Structure

- LO/: leading-order predictions from MoCaNLO
- NLOEW/: next-to-leading-order (in the electroweak coupling) predictions from MoCaNLO
- NLOQCD/: next-to-leading-order (in the strong coupling) predictions from MoCaNLO
- NNLOQCD/: next-to-next-to-leading-order (in the strong coupling) predictions from STRIPPER

## Reference

If you use these raw data please cite the following reference:

Giovanni Pelliccioli, Rene Poncelet, "Precise predictions for joint polarisation fractions in WZ production at the LHC", arXiv: 2510.25898 [hep-ph], https://inspirehep.net/literature/3075190
